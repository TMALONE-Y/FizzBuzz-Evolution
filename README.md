This repository showcases the evolution of my coding skills, comparing a script from my very first month of learning Python (October 2025) with a refactored, professional version created in April 2026.

## 📝 The Challenge
The project is a custom version of the classic **FizzBuzz**:
- **Fizz**: Multiples of 3.
- **Buzz**: Multiples of 7.
- **Almost Fizz**: Numbers that are not multiples but contain the digit '3'.

## 🛠 Improvements in the New Version
I refactored the original code to implement industry-standard practices:
- **Encapsulation**: Separated logic into `fizzbuzz(number)` and `main()` functions.
- **Error Handling**: Included `try-except` to manage non-integer user inputs.
- **Code Portability**: Added the `if __name__ == "__main__":` block for professional modularity.
